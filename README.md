# Movie Ticket Booking System

## Project Description
This is a college project developed using Python, SQLite, and Figma.
The system allows users to register, login, view movies, select show times,
and book movie tickets.

## Technologies Used
- Python
- SQLite
- Figma
- GitHub

## Features
- User Registration & Login
- Movie Listing
- Show Time Selection
- Seat Booking
- Booking Confirmation

## Team Collaboration
All team members will contribute equally to frontend, backend, database,
and documentation.

# Software Design Project

## Description
